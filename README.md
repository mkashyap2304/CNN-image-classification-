# Image Classification using CNN on CIFAR-10 Dataset

## Objective
To build a Convolutional Neural Network (CNN) to classify images into 10 categories.

## Tools Used
- Python (TensorFlow, Keras)
- Deep Learning (CNN)

## Dataset
- CIFAR-10 (60,000 images, 10 classes)

## Model Architecture
CNN with convolutional, pooling, dense, dropout, and batch normalization layers.

## Model Performance
- Training Accuracy: ~88%
- Validation Accuracy: ~72%
- Test Accuracy: ~70%

## Key Insights
- CNN performs well on image classification tasks
- Overfitting observed due to limited image size
- Similar classes are harder to classify

## Business Impact
- Can be used in image recognition systems
- Useful for automation and AI-based classification

## Conclusion
CNN models can effectively classify images, but performance can be improved using advanced architectures and techniques.
